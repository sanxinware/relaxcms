快速安装

在线安装

当前发布的版本支持在线一键安装，要求服务器能访问到RelaxCMS官网。 把以下命令复制到LINUX的SHELL环境下执行一键安装命令。默认自动检测CRAB运行环境是否安装，如果未安装，则自行先下载安装CRAB运行环境，再安装RelaxCMS框架。

if [ -f /usr/bin/curl ];then curl -o install.sh -sSLO https://www.relaxcms.com/onekeyinstall;else wget -O install.sh https://www.relaxcms.com/onekeyinstall;fi;bash install.sh
  • 注1：当前发布的版本仅支持在LINUX的SHELL环境下执行一键安装命令
  • 注2：默认安装RelaxCMS官网发布的最新稳定版本

下载安装

  • 安装准备

    安装前准备好RelaxCMS运行所依赖的PHP运行环境，如LAPM、CRAB; 推荐使用CRAB搭建服务器环境。

  • 下载安装包

    从RelaxCMS官网下载RelaxCMS所需版本，安装包格式：relaxcms-.tar.gz

  • 解压安装包

    解压安装包到已经备好的CRAB/LAPM环境的文档根目录（DocumentRoot)或虚拟主机根目录，CRAB如：/opt/crab/var/www

  • 注：CRAB环境下解压后，支持通过./setup.sh命令安装到/opt/crab/var/www
